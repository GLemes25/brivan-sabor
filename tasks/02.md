Aqui estão as informações do novo projeto (Brivan Sabor) para você adaptar o conteúdo:

- **O que é:** Um e-commerce de uma empresa de gastronomia especializada em salgados, doces, tábuas de frios e buffet para eventos.
- **Design System:** Luxury Design (fundo escuro "brand-black", tipografia dourada "brand-gold", estética impecável focada em mobile-first).
- **Stack:** Next.js (App Router), React, TypeScript, Tailwind CSS, Framer Motion, pnpm.
- **Funcionalidades principais a destacar:** Catálogo de produtos dividido por categorias, carrinho de compras (sacola), integração direta com WhatsApp para pedidos e design componentizado focado em conversão.
- **Destaques Técnicos:** Refatoração estrita de código gerado no Figma para arquitetura moderna do Next.js (Server/Client components), otimização pesada de imagens para carregamento rápido no mobile e regras de linting/arquitetura severas (código em inglês, interface em pt-BR).

## 🚀 Como Executar o Projeto

